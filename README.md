# 🚀 Smart Study Manager Pro

Ứng dụng **Smart Study Manager Pro** là hệ thống web hỗ trợ sinh viên quản lý học tập một cách trực quan, hiện đại và hiệu quả.

---

## 🎯 Mục tiêu

- Quản lý môn học và công việc học tập
- Theo dõi deadline và tiến độ hoàn thành
- Thực hành DevOps: Git, Docker, Docker Compose
- Triển khai hệ thống Backend + Frontend + Database

---

## 🧩 Công nghệ sử dụng

### Backend
- Node.js
- Express
- MongoDB (Mongoose)

### Frontend
- React (Vite)
- Axios
- React Router

### DevOps
- Docker
- Docker Compose
- Git & GitHub
- Docker Hub

---

## ⚙️ Tính năng chính

- ✅ Thêm việc học
- 🔄 Cập nhật trạng thái (Chưa làm / Đang làm / Hoàn thành)
- ❌ Xóa việc học
- 🔍 Tìm kiếm và lọc
- 📊 Thống kê tiến độ
- 📅 Hiển thị deadline (cảnh báo sắp hết hạn)
- 👤 Trang thông tin sinh viên `/about`
