# Smart Study Manager

Ứng dụng quản lý học tập thông minh với:
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB
- Docker + Docker Compose

## Chạy bằng Docker
```bash
docker compose up --build
```

## Truy cập
- Frontend: http://localhost:5173
- About: http://localhost:5173/about
- Health: http://localhost:5000/health
